译通非遗·非遗英译专业辞典软件 V1.0
项目概述
本软件是一款面向非物质文化遗产领域的专业英汉翻译词典工具，由个人独立开发完成，核心功能为非遗文化术语的英汉互译查询、关键词检索与词条展示，旨在为非遗文化的国际传播、学术研究与翻译实践提供标准化的术语支持。
核心功能
1. 非遗词条检索：支持中文/英文关键词模糊搜索，快速匹配1000+非遗专业术语
2. 词条展示：清晰呈现非遗术语的中文名称、英文翻译、术语说明
3. 响应式适配：兼容PC端/移动端浏览器，适配不同设备访问
4. 纯公益属性：无广告、无付费、无用户数据收集，面向全球用户免费开放
技术栈
前端：HTML5 + CSS3 + JavaScript
部署：GitHub Pages
开发环境：Chrome/Edge浏览器、VS Code
